A-Lab Software 18 FREE Skyboxes - Volume 1
------------------------------------------

Thank you for downloading this UnityPackage containing 18 FREE skyboxes for your use in your projects both non-commerical and commerical.

Special thanks to 3delyvisions.com for providing the beautiful skybox textures, if used commerically please give credit to 3delyvisions.


INSTALLATION

1. Extract the A-Lab_Software_Skyboxes_18_Vol_1_FREE.unitypackage file to your computer.

2. Open Unity, and in the Project view, right-click and select Import Package..., select the A-Lab_Software_Skyboxes_18_Vol_1_FREE.unitypackage file and click Open.

3. The package will then decompress, and finally prompt for confirmation to import, simply click the Import button.

4. To use in Unity, simply click the Edit menu, and select Render Settings, then in the Inspector pane select any of the 18 skybox materials, named Skybox1 to Skybox18.

5. Finally, add the MouseLook script from the Standard Assets\Camera Scripts to the Main Camera in the Hierarchy pane, and click Play to view the scene.
